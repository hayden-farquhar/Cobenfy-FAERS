# Data

## FAERS Quarterly ASCII Files

Raw FAERS data files are not included in this repository due to their size (~50 GB total). To reproduce the analysis, download the quarterly ASCII files from the FDA:

**Download URL:** https://fis.fda.gov/extensions/FPD-QDE-FAERS/FPD-QDE-FAERS.html

### Required quarterly files

Download the **ASCII** format (not XML) for each of the following quarters:

| Quarter | File | Approximate size |
|---|---|---|
| Q4 2024 | faers_ascii_2024Q4.zip | ~10 GB |
| Q1 2025 | faers_ascii_2025Q1.zip | ~10 GB |
| Q2 2025 | faers_ascii_2025Q2.zip | ~10 GB |
| Q3 2025 | faers_ascii_2025Q3.zip | ~10 GB |
| Q4 2025 | faers_ascii_2025Q4.zip | ~10 GB |

### Automated download

Script `01_download_faers.py` will attempt to download and extract these files automatically:

```bash
python scripts/01_download_faers.py
```

This places zip files in `data/raw/` and extracted CSVs in `data/raw/parsed/`.

### Data tables

Each FAERS quarterly file contains seven tables:

| Table | Description | Key fields |
|---|---|---|
| DEMO | Demographics | primaryid, caseid, age, sex, reporter country |
| DRUG | Drug information | primaryid, drugname, role_cod, dose_amt |
| REAC | Reactions (adverse events) | primaryid, pt (MedDRA preferred term) |
| OUTC | Outcomes | primaryid, outc_cod (DE/HO/LT/DS/CA/RI/OT) |
| INDI | Indications | primaryid, indi_pt |
| THER | Therapy dates | primaryid, start_dt, end_dt |
| RPSR | Report sources | primaryid, rpsr_cod |

### Deduplication

FAERS contains duplicate reports (same case reported multiple times across quarters). Script `02_load_deduplicate.py` deduplicates by CASEID, retaining the most recent report version. The resulting DuckDB database contains 1,808,859 unique cases.

### Drug name standardisation

Script `03_drug_standardisation.py` maps free-text drug names to standardised active ingredients. Xanomeline-trospium is identified through six name variants:

- XANOMELINE AND TROSPIUM CHLORIDE
- XANOMELINE/TROSPIUM
- COBENFY
- KARXT
- XANOMELINE-TROSPIUM
- XANOMELINE TROSPIUM

Active comparators (olanzapine, risperidone, aripiprazole, quetiapine, lurasidone, brexpiprazole) are similarly standardised.
